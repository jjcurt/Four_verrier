ℹ️ Format des programmes

Les programmes sont stockés au format JSON avec des étapes (steps) définissant les paramètres de cuisson.

Champs requis pour chaque étape :

  • targetTemp: Température cible (°C)
  • rampRate: Vitesse de montée (°C/min, 0 = palier uniquement)
  • withRamp: true si montée en température, false sinon
  • holdTime: Durée de maintien (minutes)
  • useSSR2: true pour utiliser SSR2, false sinon

Exemple :

{"name":"Test","numSteps":2,"steps":[
  {"targetTemp":500,"rampRate":100,"withRamp":true,"holdTime":30,"useSSR2":false},
  {"targetTemp":0,"rampRate":50,"withRamp":false,"holdTime":0,"useSSR2":false}
]}
